# PalindromeCheckerApp

## UC4: Character Array-Based Palindrome Check
**Goal:** Convert string to `char[]` and compare characters.  

**Flow:**  
- Convert string to char array.  
- Use two-pointer approach.  
- Compare start and end characters.  

**Key Concepts:** Character arrays, indexing, two-pointer technique, time complexity awareness.  
**Data Structure:** `char[]`.
